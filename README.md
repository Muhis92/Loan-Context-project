# Loan Context Project

## Overview
This project is a React application built to practice using the Context API for global state management.

The app demonstrates how to share data between multiple components without prop drilling.

---

## Features
- Global state management using React Context
- Multiple reusable components
- Data sharing between components
- Clean component structure
- Simple UI with modular design

---

## Technologies Used
- React
- JavaScript (ES6+)
- CSS

---

## Project Structure
src/
  components/
  context/
    InputContext.js
  App.js
  index.js

---

## Getting Started

1. Install dependencies:
npm install

2. Run the project:
npm start
